Acerca de esta app

Venezuela InfoSismo es una aplicación de código abierto, no oficial, creada por ciudadanos para facilitar el acceso a información pública durante la emergencia sísmica de 2026.

Esta app no almacena datos personales, no tiene backend propio y únicamente agrega información de fuentes públicas verificables.

Los datos sísmicos provienen de USGS (United States Geological Survey), una fuente oficial y pública.

Aviso importante

Verifica SIEMPRE la información con las autoridades oficiales. En caso de emergencia activa, llama al 911 o a Protección Civil.
