Acerca de esta app

Venezuela InfoSismo es una aplicación de código abierto, no oficial, creada para facilitar el acceso a información pública durante la emergencia sísmica de 2026.

Esta app no almacena datos personales, no tiene backend propio y únicamente agrega información de fuentes públicas verificables.
